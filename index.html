🔬 #01

Grant Opportunity Matchmaking Agent

84% match similarity · Top-10 faculty per grant
Agentic AI Generative AI LangChain LangGraph pgvector PostgreSQL AWS Bedrock Claude Haiku Python Vector Search
View on GitHub

Identifying relevant research funding opportunities across hundreds of grant announcements is a time-consuming challenge for faculty. This project addresses that by building an agentic AI workflow orchestrated with LangGraph, where LLM reasoning steps handle the full pipeline — from ingesting faculty profiles and extracting CV publications, to retrieving matching grants and generating detailed match justifications. LangChain powers the retrieval and reasoning layers, making the entire process automated and explainable.

Multi-stage retrieval pipeline combining pgvector ANN search with LLM reranking, delivering similarity scores up to 84% with top-10 faculty recommendations per grant — with end-to-end latency reduced by ~70% through parallel LLM processing.
